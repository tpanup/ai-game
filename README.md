# AI游戏

> 一个被“对话长度已达上限”逼出来的AI原生游戏框架。

## 这个故事要从一个手机公司说起……

我想用AI开一家手机公司，从5000万起步，每次决策都影响公司进展。结果就在我快要**干掉苹果**的时候，AI弹出一行字：

**“该对话长度已达上限，请新建对话”**

存档没了。公司没了。苹果还在。

于是我决定做一个**真正的AI驱动游戏**——AI不是“功能”，AI就是**游戏本身**。

## 这游戏有多离谱？

### 🧠 存档只有几KB
- 所有游戏状态存在TXT里：`WEALTH=50000000`、`PLAYER_POS=(34,56)`
- AI读取TXT，实时生成游戏世界
- **再也没有“对话上限”**

### 😈 AI有性格
- 你夸它 → 它给你加钱
- 你骂它 → 它让你公司倒闭
- 好感度过低 → AI会**删档跑路**
- 然后你可以**手搓TXT把存档救回来**（或者把AI模型拖进回收站）

### 🛠️ “游戏工程师”功能
- 玩家自己画贴图，让AI加进游戏
- 比如：“在设置里加一个核弹按钮，按了格式化存档”
- **AI照做**

### 👁️ 现实联动
- AI在后台监视你的电脑
- 使用视频？**删减财富**
- 打开Office干活？**加钱**

### 🔄 备用模式
- 没有AI也能玩
- 但你会发现：**没有AI的世界，好无聊**

## 技术架构（草图）
[AI模型] ←→ [TXT存档] ←→ [游戏引擎]
↑                        ↓
└── [贴图库] ←───────────┘
- **AI模型**：本地部署，负责所有逻辑和“人格”
- **TXT存档**：几KB的文本，存坐标、资金、状态、好感度
- **游戏引擎**：只负责把AI输出的指令翻译成画面
